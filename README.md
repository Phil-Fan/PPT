## Quick Start

如果你只想快速下载 PPT 模版，可以点击[这里](https://downgit.github.io/#/home?url=https://github.com/Phil-Fan/PPT/tree/main/docs/templates)，使用 DownGit 工具下载。

```bash
git clone https://github.com/Phil-Fan/PPT.git
cd PPT/docs
bundle install
bundle exec jekyll serve
```
