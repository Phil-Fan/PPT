# Skills

## 快捷键

- `ctrl+g` 编组
- `ctrl+shift+g` 解除编组
- `ctrl+ 拖拽` 复制
- `ctrl+shift+ 拖拽` 对齐复制
- `Shift`，等比例缩放
- `ctrl+shift`，中心点等比例缩放
- `ctrl+shift+<` 减小字号
- `ctrl+shift+>` 增加字号
- 图层 `alt+f10`
